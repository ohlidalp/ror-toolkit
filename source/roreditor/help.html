How To Add Things

Classic method

- Click on terrain empty zone, a red cylinder will appear on the ground(that's the point where you will add the object)
- Rightclick on an item in the Object Tree and use Add to Terrain


Mouse method

- Open Object Tree
- select the object you desire to add
- Mark the checkbox "Place objects with Middle Mouse Button" (MMB).
- Each click with your MMB on 3D window will add a new object to the position you are pointed with your mouse.

Some objects goes sunk, so, move your camera down until you get sink and then leftclick on the object to select it, Now you can drag blue pivot or just use Object Inspector to change its "Height from Ground" property.

Because this technique is standard for all objects, some objects may be placed in middle air or sunken in terrain. To prevent this, click on an object to select it, then press F1 key (a message will be shown on screen to help you), move your object from underground to the ground until the object is in fine position, press F2 to save this offset. Now everytime you add this object to the terrain it will be on the ground instead sunked. This option is saved to disk, if you want to reset this setup, press F3 key while your object is still selected.


Block Mode method

Tipically used to build a road, it also can be used to create a big parking zone, some buildings next to the previous one, etc.

- Add the first piece of road using a previous First or Second method
- select the recent added object by Left click on it
- press SPACE key and the selected object in "Object Tree" will appear next to the selected object in 3D Window.

Free movement of objects

Although you can drag and drop each red/green/blue pivot for moving an object in the desired direction, we provided this feature that the object will follow your mouse pointed position:

- LeftClick to select an object in 3D window (maybe a context popup appear to help you to select the object you want, Leftclick in object of this popup menu)
- Keep CTRL key pressed while you leftDown again over the selected object and drag your mouse.
(if you release CTRL key or Left Mouse Button, the object will stop moving at the latest position)

Tips and Tricks

Focus, always be in of focus

Please, keep in mind that 3D windows is needed to be focused to process any key, so, before pressing a key you should rightclick on 3D window to focus it.

When precise selecting is important

If you leftclick on an object, maybe that a popup menu appears, that means that are various objects overlapped between the camera position and the point of the terrain that you are pointing at. Objects on top of the popup menu is near to the camera position (your point of view), meanwhile objects in lower part of the popup menu is near the point you are clicking on, or eventhough behind the point you are specting.

If there are only 2 objects, just left-click twice on the foreground object to select the background one.

Return key, the master key

You should press Enter key on any Text control to confirm your changes, for example, in the window Map Options, once you change the "Ingame menu map name" you must press Enter key. On Object Inspector there is a trick, you can modify X, Y, Z coordenates and then, press Enter key Once on any of the three text control box, it will update X, Y, Z coordenates in a single Step. The same happens with Rot X, Rot Y and Rot Z text Controls. On Object Comment you are allowed to press enter key to create a new line, once more time, your last key should be Enter key to update values.

Calculator

You can do some calcs on numeric text controls, for example if you have a coordenate on "1325.22" you can type something like "1325.22 + 1.58" and then press enter to evaluate that expression.

Known issues

If an object is not shown on Object Preview Toolwindow then, it has errors and Toolkit can not add this object to the terrain. Please note that the error maybe:

- File you are trying to load is newer than Toolkit File interpreter.
- Error in truck section. When RoR Toolkit detect this kind of errors, a red label will be displayed in Object Preview toolwindow, you can check console for further help.
- Toolkit errors when it interpretes files.
